# AI Interview (AI Mock Interview App)

AI Interview is a web-based mock interview platform that helps candidates practice interviews in a realistic flow and receive structured, AI-assisted feedback. It is designed to be easy to understand for **HR/Recruiters** (what it does and why it’s useful) and for **Engineering teams** (what technologies and services are used).

---

## What this project does

- Allows a user to run **mock interview sessions**
- Presents **role/skill-based interview questions**
- Collects candidate responses during the session
- Generates **AI-assisted evaluation/feedback** to highlight:
  - strengths
  - improvement areas
  - overall readiness (based on the implemented scoring/feedback logic)
- Supports common “interview prep” workflows such as repeating sessions and improving over time (depending on what is implemented in the UI/data layer)

## Tech stack / Tools used (from `package-lock.json`)

### Frontend & Framework
- **Next.js** (`next` ^15.5.1-canary.20) — React framework used to build the app
- **React** (`react` 19.1.0) + **React DOM** (`react-dom` 19.1.0)
- **TypeScript** — primary language for the codebase

### Styling & UI
- **Tailwind CSS** (`tailwindcss` ^4) — utility-first styling
- **tailwind-merge** — class merging utility for Tailwind
- **tw-animate-css** — animation helpers for UI
- **Radix UI** (`@radix-ui/react-dialog`, `@radix-ui/react-tabs`, `@radix-ui/react-slot`) — accessible UI primitives
- **Icons**
  - `lucide-react`
  - `@tabler/icons-react`
- **sonner** — toast/notification UI

### Authentication
- **Clerk** (`@clerk/nextjs`) — authentication and user management (sign-in, sign-up, session handling)

### Backend / Data / Serverless
- **Convex** (`convex`) — backend/database + server functions (depending on how you configured it)

### Networking / API
- **axios** — HTTP client for calling APIs

### Security / Protection
- **Arcjet** (`@arcjet/next`, `@arcjet/inspect`) — request protection / security middleware (rate limiting / bot protection depending on configuration)

### Files, Media & Documents
- **ImageKit** (`imagekit`) — image/file storage and delivery (if wired in)
- **react-dropzone** — drag-and-drop file upload UI
- **pdfjs-dist** + `@types/pdfjs-dist` — PDF viewing/processing support

### AI / Avatar (if enabled in the app)
- **akool-streaming-avatar-sdk** — streaming avatar integration (for an AI avatar/interviewer experience)
- **motion** — animation library (used for UI transitions and interactive motion)

### Developer Tooling
- **@types/node**, **@types/react**, **@types/react-dom** — TypeScript typings
- **typescript** — TypeScript compiler

## Getting started (Run locally)

### Prerequisites
- **Node.js 18+** is recommended (several dependencies require Node >= 18)

### Setup
```bash
git clone https://github.com/Utkarsh-2711/ai-interview.git
cd ai-interview
npm install
npm run dev
```

### Production build
```bash
npm run build
npm run start
```
